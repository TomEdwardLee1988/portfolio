Settings

You can change the length of time that the sentence strips and correct matches, and any incorrect matches remain on screen.
You will need to change this every time you start a new game (including resets), or the times will return to their default settings.

How to play

The aim of the game is to find all the matching cards. Begin by selecting a card, it will flip over to show a picture - click on another card to try and find the matching pair!

You can change how long the cards stay on the screen in the settings window, but please be aware settings will return to default whenever you start or refresh a game.
